# ⚡️Mobile

Grid works on mobile!

Make sure to save the app on your home screen!

## iOS

1. Open safari
2. Go to [platform.grid.ai](https://platform.grid.ai)
3. Tap the arrow at the bottom
4. Tap "Add to home screen"
5. Tap done
